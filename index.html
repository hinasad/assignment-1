Italic

How to Italicize Text in HTML. To italicize the text in HTML, use either the em tag or the i (italics) tag. Both of these tags will italicize the text,
but the em tag additionally indicates that the text has stress emphasis when read. You can also italicize text with the CSS font-style property set to “italic.”

Bold

To bold the text in HTML, use either the strong tag or the b (bold) tag. Browsers will bold the text inside both of these tags the same, but the strong
indicates that the text is of particular importance or urgency. You can also bold text with the CSS font-weight property set to “bold.”
